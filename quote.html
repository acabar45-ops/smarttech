🙋 사용 설명서 — 쉽게 풀어쓰기

이 프로그램은 고객이 보낸 이메일을 붙여넣기만 하면 AI가 견적서를 자동으로 만들어 주는 도구예요. 마치 똑똑한 신입사원이 "이 메일 보고 견적서 대신 써줘!" 하면 바로 만들어주는 것과 같아요. 아래를 천천히 읽으시면 누구나 5분 안에 쓸 수 있습니다.

⚡ 1분 만에 끝내는 사용법

  1. ① 메일 탭 열기 → 고객이 보낸 메일을 전부 복사해서 붙여넣기
  2. [에이전트 분석 시작] 버튼 누르기 (10~20초 기다림)
  3. AI가 알아서 고객 등급·제품 목록·답장 초안을 만들어 줌
  4. [적용 · 견적서 작성으로] 버튼 → ② 견적 탭으로 자동 이동
  5. ③ 미리보기 탭 → 위쪽에 답장 초안이 있으니 [📋 복사] 눌러서 Gmail에 붙여넣기
  6. 같은 화면에서 [인쇄 / PDF 저장] → 견적서는 PDF로 첨부

실제 예시: 고객이 "안녕하세요, 동결건조기에 쓸 nXDS15i 펌프 2대 견적 부탁드립니다. 납기 언제 가능한가요?" 같은 메일을 보내면 → AI가 "최종사용자 등급이네요" 판단 → nXDS15i 펌프 2대를 장바구니에 담고 → "안녕하세요, ○○님. 문의 주신 nXDS15i 2대 견적서 첨부드립니다. 납기는 재고 보유 시 1~2주..." 같은 답장까지 자동으로 써줍니다.

📋 2 · 각 메뉴 설명 (예시와 함께)

① 메일 자동견적 — 가장 먼저 쓰는 메뉴

고객이 "펌프 견적 보내주세요" 라고 보낸 이메일을 붙여넣으면, AI가 3명의 직원처럼 나눠서 일해줘요:

  • 첫 번째 AI (등급 판별) — "이 고객이 딜러인지? 제조사인지? 일반 사용자인지?" 를 메일 내용으로 판단. 예) "재판매 하겠다"라고 쓰여 있으면 → 딜러, "저희 연구소에서 쓸 거예요" → 최종사용자
  • 두 번째 AI (제품 찾기) — 914개 제품 카탈로그에서 고객 요청에 맞는 걸 골라서 장바구니에 넣어줘요. 예) "nXDS15i" 라고 적혀 있으면 정확히 그 제품을 찾고, "동결건조용 펌프"라고만 적혀 있으면 맞는 펌프 5~8개를 추천
  • 세 번째 AI (답장 쓰기) — "안녕하세요 ○○님, 문의 주셔서 감사합니다..." 로 시작하는 정중한 답장을 자동으로 써줘요. 고객 질문(납기·오일 포함 여부 등)에 하나씩 답해주고, 확답 어려운 건 "확인 후 안내드리겠습니다" 처리.

⏱ 보통 10~20초 걸려요. 결과가 나오면 [적용 · 견적서 작성으로] 버튼 누르기.

② 견적서 작성 — 내용 확인·수정

AI가 만들어준 내용을 눈으로 한번 더 확인하는 화면이에요. 바꾸고 싶은 게 있으면 여기서 바꾸세요.

  • 고객 등급 — 딜러/OEM/최종사용자 중 선택. 등급에 따라 가격이 달라져요. 예) 딜러는 싸게, 최종사용자는 정가대로
  • % 입력칸 — 각 등급 옆의 숫자는 "기본 가격의 몇 퍼센트로 팔까?" 예요. 기본 100%. 예) 특별히 3% 더 할인하려면 97 입력
  • 견적 미세 조정 — 이번 견적만 전체적으로 깎고 싶을 때 쓰는 슬라이더. 예) 전체 5% 할인이면 95%로 맞춤
  • 명함 스캔 — 고객 명함 사진 찍어 올리면 회사·이름·전화·이메일이 자동 입력 (메일 받은 경우는 이미 채워져 있음)
  • 수량·단가 직접 수정 — 숫자 클릭해서 바로 바꾸기. 합계는 자동 계산

③ 미리보기 · 출력 — 마지막 확인과 보내기

실제 A4 종이에 인쇄될 견적서가 보여요. 위쪽에는 📩 고객이 보낸 메일과 ✉ AI가 쓴 답장 초안이 나란히 떠요.

  • [📋 회신 복사] 버튼 → Gmail/아웃룩에서 답장 쓰기 열고 Ctrl+V (붙여넣기)
  • [인쇄 / PDF 저장] 버튼 → 프린터 대화창 → "PDF로 저장" 선택 → 견적서 PDF 파일 완성
  • PDF 파일을 Gmail 답장창에 첨부하면 끝! (답장 본문은 ✉ 복사한 내용 그대로 두면 됨)
  • 💡 팁: 인쇄할 때 원본 메일과 답장 박스는 자동으로 숨겨져요. 견적서만 깔끔히 출력됩니다.

④ 품목 검색 — 메일 없이 직접 찾을 때

이메일로 안 왔는데 급하게 견적 만들어야 할 때 씁니다. 제품 이름이나 번호(예: RV3, GXS160) 넣어 검색 → 찾으면 [담기] 버튼으로 장바구니 추가.

⑤ 공정별 추천 — 고객이 "어떤 펌프가 좋을까요?" 물을 때

9개 공정 중 선택 (코팅, 이차전지, 진공로, 가스실린더, 이중배관, 동결건조, OLED, 진공오븐, 연구/분석). 각 공정에 어울리는 펌프 모델을 자동으로 보여줘요. 예) "동결건조" 선택 → RV·E2M·GXS 시리즈 추천

⑥ 고객 관리 — 주소록 같은 것

한번 거래한 고객 정보를 저장해두고, 다음에 견적 만들 때 이름만 클릭하면 회사·담당자·전화가 자동 입력돼요.

⑦ 발행 내역 — 지난 견적 보관함

보낸 견적이 자동으로 저장됩니다. 과거 견적을 다시 열거나, 같은 내용으로 약간 수정해서 재발행할 수 있어요. 예) "3개월 전 ○○전자에 보낸 견적이랑 똑같이 다시 보내야 해" → 발행 내역에서 해당 건 [불러오기]

⑧ 설정 — 가격 정책 조정

등급별 할인율을 항상 바꿔 쓰고 싶을 때 여기서 조정. [수정하기] → 값 변경 → [저장하기] 순서로 누르세요. 예) 딜러에게는 항상 95%로 팔고 싶다면 딜러 칸을 95로 고정

💰 3 · 가격은 어떻게 계산되나요?

구구단 같은 순서로 곱셈만 하면 끝이에요. 예시로 따라가 봅시다.

1단계 엑셀 원본 가격 — 등급에 따라 다름. 예) 어떤 제품의 딜러가 = 100,000원
2단계 등급 할인율 곱하기 — 설정탭에서 딜러를 95%로 해뒀다면: 100,000 × 95% = 95,000원
3단계 이번 견적만 미세 조정 곱하기 — 슬라이더를 98%로 뒀다면: 95,000 × 98% = 93,100원 ← 이게 한 개 단가
4단계 수량 곱하기 — 2대 사면: 93,100 × 2 = 186,200원
5단계 부가세 10% 추가 — 186,200 + 18,620 = 204,820원 (총 공급가액)

기억할 점:

  • 설정 탭의 % (등급 배수) = "앞으로 모든 견적에 이 할인율 쭉 쓸게요" → 저장하면 계속 적용됨
  • 견적서 작성 탭의 % (미세 조정) = "이번 견적만 특별히 더 할인/인상" → 다른 견적에는 영향 없음
  • 단가를 직접 숫자로 바꾸면 위 공식 무시하고 그 값으로 계산됩니다

🤖 4 · AI는 어떻게 일하나요?

메일 자동견적 버튼을 누르면 AI 3명이 차례로 일해요. 쉽게 비유하면 신입사원 3명이 역할 나눠서 도와주는 것과 같아요.

🕵️ AI 1번 — 고객이 어떤 분인지 파악

메일을 읽고 "이 고객은 딜러일까? 제조사일까? 일반 사용자일까?" 를 판단해요. 동시에 회사명·담당자 이름·직급·전화번호·이메일을 메일에서 뽑아냅니다.

판단 예시:
  • "저희가 재판매하려 합니다" → 딜러
  • "저희 장비에 넣을 부품이 필요합니다" → OEM (제조사)
  • "저희 연구소에서 쓸 거예요" → 최종사용자
  • 애매하면 안전하게 최종사용자로 기본 판정 (가장 비싼 가격대)

🎯 AI 2번 — 알맞은 제품 찾기

914개나 되는 제품 중에서 고객 요청에 맞는 걸 찾아줘요. 그냥 다 보여주면 AI가 헷갈리니까, 프로그램이 먼저 "이 메일엔 이 계열 제품이 맞을 것 같아" 라고 후보를 40~80개로 줄인 다음 AI에게 넘깁니다.

찾는 방법:
  • 메일에 정확한 모델명(nXDS15i, GXS450 등) 있으면 → 그걸 최우선 매칭
  • 공정만 쓰여 있으면 → 그 공정에 맞는 시리즈를 추천 (예: "동결건조" → RV·E2M 계열)
  • 수량이 써 있으면 그대로, 없으면 1대
  • 카탈로그에 없는 제품은 절대 만들어내지 않음 (환각 방지)

✉ AI 3번 — 답장 초안 쓰기

AI 1·2가 모은 정보로 고객에게 보낼 답장을 자동으로 써요. 인사 → 감사 → 고객 질문에 답변 → 견적 요약 → 다음 단계 안내 → 스마텍 서명 순서로 500~900자.

❓ 5 · 자주 묻는 질문

  • Q. AI가 잘못 판단하면요?
    A. 모든 결과는 눈으로 확인 후 수정 가능해요. 답장 초안도 마음에 안 들면 [재생성] 누르거나 직접 편집하세요. 등급도 잘못 판정했다 싶으면 견적서 작성 탭에서 바로 바꾸면 됩니다.
  • Q. 인터넷 없이도 돼요?
    A. 기본 기능(제품 검색·견적서 만들기·인쇄)은 인터넷 없어도 돼요. 단 AI 기능(메일 자동견적·명함 스캔)은 인터넷 필요합니다.
  • Q. 회사 이름이 안 뜨는데?
    A. 상단에 비밀번호 8245 넣고 [로그인] 누르셨는지 확인. 로그인해야 고객·견적 내역이 저장되고 AI 기능도 작동합니다.
  • Q. 핸드폰에서도 돼요?
    A. 네. 핸드폰에서는 하단에 ① 메일 · ② 견적 · ③ 미리보기 · ⋯ 더보기 버튼이 보여요. 순서대로 누르시면 됩니다.
  • Q. 견적 보낸 거 다시 찾으려면?
    A. 발행 내역 탭에서 날짜·회사명으로 검색 → [불러오기] 누르면 그대로 다시 열립니다.
  • Q. PDF 저장하는 법?
    A. 미리보기 화면에서 [인쇄] 버튼 → 프린터 대화상자에서 프린터 대신 "PDF로 저장" 선택 → 파일 이름 정하고 저장.

🛟 6 · 문제 해결

  • AI 분석이 안 돼요 — 상단에서 로그아웃되어 있는지 확인. 비번 8245로 재로그인.
  • 429 Rate Limit 에러 — 너무 자주 요청해서 잠시 차단된 상태. 1분 기다렸다 재시도.
  • 미리보기에서 답장이 안 보여요 — 메일 탭에서 [적용 · 견적서 작성으로] 버튼을 꼭 눌러야 답장이 미리보기로 전달돼요.
  • 화면이 이상해요 — 브라우저에서 Ctrl+Shift+R(강제 새로고침) 한번 눌러주세요.
  • 버튼이 안 눌려요 — 페이지 한번 껐다 다시 열어주세요.
